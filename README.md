# app_caisse

Application de caisse Flutter.

## 🎯 Objectif

Fournir une solution de caisse moderne, rapide et fiable pour gérer :
- Les catégories et sous-catégories d’articles
- Les prix fixes et prix variables
- Les tickets en temps réel
- Les paiements (Espèces, CB, Chèque)
- Le calcul automatique du rendu
- La sauvegarde locale des tickets

## 🛠️ Technologies

- **Flutter** (Dart)
- **Windows** (EXE)
- **Android** (APK)
- Architecture modulaire (models, services, pages, widgets)
